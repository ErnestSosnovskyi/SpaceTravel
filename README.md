# SpaceTravel - Ticket Booking System Backend

Цей проєкт реалізує архітектуру бази даних для міжпланетної транспортної компанії **SpaceTravel** з використанням Java, Hibernate та міграцій Flyway.

## Стек технологій
* Java 17+
* Gradle
* Hibernate ORM 6
* Flyway (Database migrations)
* H2 Database (File-based mode)
* Lombok

## Як запустити проєкт
1. Клонуйте репозиторій.
2. Переконайтеся, що у вас встановлена Java 17 або вище.
3. Виконайте команду для запуску додатку через Gradle:

```bash
./gradlew run
```

## Опис структури даних

* Client: Сурогатний автоінкрементний ID, ім'я від 3 до 200 символів.
* Planet: Рядковий ID (тільки великі літери й цифри, наприклад MARS), назва від 1 до 500 символів.
* Ticket: Містить часову мітку створення квитка (created_at) у UTC, та зв'язки ManyToOne з клієнтом та планетами відправлення/прибуття.
